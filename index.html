Welcome to the World of Padman Archive!

The World of Padman Archive is a collection of content that was originally hosted at worldofpadman dot com until the crash of the webserver in ca 2014, and aims to preserve it for the afterworld.

Contributing

WoP archive

  • You have old official releases (pre PadMod 2004 maps and models, EliteForce & JediKnight2 PadMaps) that are missing from the downloads page?
    Please create a new issue and provide a download link so that these can be archived.
  • For content fixes (grammar, styles etc.) please fork the repository, commit your fixes and create a new pull request.
  • If you'd like to provide a http/ftp mirror for the downloads, please create a new issue with a link to the mirror. (Unis welcome!)
